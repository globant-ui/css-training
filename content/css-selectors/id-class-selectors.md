# ID and Class selectors

## Description

Simple selection by id and class

## Code

```html
<!--ID selector-->
<div id="myRedBox"></div>

<!--class selector-->
<div class="my-blue-box"></div>
```

```css
/*id selector*/
#myRedBox{
  width: 100px;
  height: 100px;
  background: #f00;
}

/*class selector*/
.my-blue-box{
  width: 100px;
  height: 100px;
  background: #00f;
}
```
